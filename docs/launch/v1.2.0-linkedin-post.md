## Post body

```
aiPal v1.2 is out — a self-hosted personal AI assistant built with Laravel.

Here's what's new since v1.0:

🤖 Agent Workflows
Schedule any prompt, trigger it from a webhook, or wire it to a WhatsApp/Telegram message prefix. "Every Monday 8am, summarize last week's Jira tickets and send to WhatsApp." Works.

📍 Location + Weather
Captures your location from the browser, a native WhatsApp share, or a pasted Google Maps link. Ask "will it rain on my run tomorrow?" — no city typing.

🔍 Unified Search (Cmd+K)
One shortcut searches every memory, note, task, document, and past conversation.

🐙 GitHub + GitLab code review
AI reviews your PRs and posts inline comments back on the platform.

📲 PWA + Web Push
Installs like a native app on iPhone and Android. Push notifications for reminders actually fire.

⚙️ Fully provider-agnostic
Every model call — chat, memory, embeddings, STT, TTS, daily briefing — is swappable via .env. Claude for one thing, local Ollama for another. No lock-in.

Still in the box from v1.0: multi-provider chat, semantic memory (pgvector), RAG over your documents, voice in/out, tasks/reminders/notes, Google Calendar + Gmail, Telegram + WhatsApp bots, fully offline mode via Ollama.

Stack: Laravel 13 · PHP 8.4 · pgvector · Redis · Livewire · Tailwind v4 · Docker
AGPL-3.0. Runs on a Raspberry Pi.

Link in the first comment 👇

What would you want your AI assistant to do that no SaaS tool does today?

#Laravel #PHP #OpenSource #AI #SelfHosted #BuildInPublic
```
